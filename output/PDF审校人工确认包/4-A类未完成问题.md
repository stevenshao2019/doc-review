## 批量处理速查表

以下为排版软件（InDesign/Word）全局替换代码，可一次性解决约70%的剩余空格问题：

```regex
# 替换规则（正则表达式）

# 中文引号内侧空格（左）
查找："\s+([^"]*)
替换："$1

# 中文引号内侧空格（右）— 谨慎，需区分左右引号
查找：([^"]*)\s+"
替换：$1"

# 左括号后空格
查找：\(\s+
替换：(

# 右括号前空格
查找：\s+\)
替换：)

# 间隔号前后空格
查找：·\s+
替换：·
查找：\s+·
替换：·

# 斜杠前后空格
查找：\s+/\s+
替换：/

# 连字符前后空格（英文复合词）
查找：\s+-\s+
替换：-

# 数字与"年""年代"间空格
查找：(\d+)\s+年
替换：$1年
查找：(\d+)\s+年代
替换：$1年代

# 特殊拼写替换（逐一替换，非正则）
Boszormenyi-Nagi → Boszormenyi-Nagy
Tuscon → Tucson
汨汨 → 汩汩
```
